# Workout Log

Projeto para registrar histórico de atividades físicas

### Desenvolvimento

```sh
git clone git@github.com:samuelpadua/workout.git
cd workout
npm install && npm start
```

### Testes

```sh
npm test
npm run test:watch
npm run test:cov
```

### Eslint
```sh
npm run lint
```


### Construido com
* React
* Redux
